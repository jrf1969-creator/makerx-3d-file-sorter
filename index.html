STP

Format Not Supported

STEP / STP files use a complex CAD geometry kernel that can't be parsed directly in the browser without a large (~50 MB) engine download.

Convert your file to STL or 3MF first — both are fully supported here.

FreeCAD (Free) Onshape (Free) CAD Exchanger ConvertCADFiles

In FreeCAD: File → Open your STEP → File → Export → select STL or 3MF.
In Onshape: Import STEP → right-click part → Export → STL.
